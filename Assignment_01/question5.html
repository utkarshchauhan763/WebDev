What is a Web Server?

A web server is software (and often the machine it runs on) that listens for HTTP requests from clients (browsers or other agents), processes those requests, and returns HTTP responses (HTML, JSON, files). It routes requests to appropriate handlers and may serve static files or invoke application logic to produce dynamic responses.

Common Web Servers

  • Apache HTTP Server — widely used, flexible, supports modules and .htaccess configuration.
  • Nginx — high-performance, often used as a reverse proxy, load balancer, or static file server.
  • Microsoft IIS — web server for Windows environments.
  • Node.js (Express) — not a traditional web server but used to build application servers handling HTTP via frameworks like Express.
  • Caddy — modern web server with automatic HTTPS (LetsEncrypt) by default.
