Static vs Dynamic Websites

Static Websites

Static websites serve pre-built HTML files to clients. Content does not change per request unless the files are changed on the server. They are simple, fast, and easy to host (e.g., via GitHub Pages).

Example: A brochure site for a local business with a few pages (Home, About, Contact) that rarely change.

Dynamic Websites

Dynamic websites generate responses on the server per request, often using server logic and databases. Pages can be personalized or updated frequently.

Example: An e-commerce site where product availability, user carts, and recommendations are generated dynamically from a database.
