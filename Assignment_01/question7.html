Install VS Code and Configure for Web Development

Installation (Windows)

  1. Download the installer from: https://code.visualstudio.com/
  2. Run the downloaded .exe and follow the installer steps.
  3. Optionally check "Add to PATH" and "Open with Code" during installation.

Basic Configuration and Extensions

  1. Open VS Code.
    • File > Preferences > Settings to adjust editor font size, tab size, and default formatter.
  2. Install recommended extensions:
    • HTML CSS Support (built-in HTML features are strong already)
    • ESLint — lint JavaScript
    • Prettier — code formatter
    • Live Server — serve pages locally with reload
  3. Configure default formatter (if using Prettier):
    • Settings > Default Formatter > select esbenp.prettier-vscode
  4. Open a project folder and use the integrated terminal (View > Terminal) for running commands.
