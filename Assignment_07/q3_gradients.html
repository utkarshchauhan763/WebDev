Gradient Background Demonstration

Exploring Linear and Radial Gradients with CSS

Linear Gradient Background

This section uses a linear gradient that transitions smoothly from one color to another in a straight line.

CSS Property: background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

Linear gradients can be customized with different angles, multiple color stops, and opacity values to create stunning visual effects.

Radial Gradient Background

This section features a radial gradient that radiates from a central point outward in a circular or elliptical pattern.

CSS Property: background: radial-gradient(circle, #f093fb 0%, #f5576c 100%);

Parallax Effect: This section uses background-attachment: fixed; to create a parallax-like scrolling effect.

Radial gradients are perfect for creating spotlight effects, vignettes, and eye-catching backgrounds.

Gradient Applications

CSS gradients are powerful tools for modern web design. They can be used for:

  • Background colors with smooth transitions
  • Creating depth and dimension
  • Button and card styling
  • Overlays and masking effects
  • Text backgrounds with background-clip
