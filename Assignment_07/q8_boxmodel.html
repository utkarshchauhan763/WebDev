CSS Box Model Exploration

Understanding content, padding, border, and margin

Default Box Model (content-box)

box-sizing: content-box (default)
Width/height applies only to content. Padding and border are added on top.

CONTENT

Width: 200px

Height: 150px

Total Width: 200px (content) + 40px (padding) + 6px (border) = 246px
Total Height: 150px (content) + 40px (padding) + 6px (border) = 196px

Border-Box Model

box-sizing: border-box
Width/height includes content, padding, AND border. More predictable sizing.

CONTENT

Width: 200px

Height: 150px

Total Width: 200px (includes everything)
Total Height: 150px (includes everything)
Content area shrinks to fit padding and border inside the specified dimensions.

CONTENT
  • Content: The actual content (text, images, etc.)
  • Padding: Space between content and border (inside the element)
  • Border: The border surrounding the padding and content
  • Margin: Space outside the border (separates from other elements)
